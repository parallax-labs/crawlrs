# Crawlrs

A web crawling tool written in rust

```sh
crawlrs 0.1.0

USAGE:
    crawl [FLAGS] [OPTIONS]

FLAGS:
    -h, --help           Prints help information
        --json           output json
    -s, --same-domain    Only crawl pages from the same domain
    -V, --version        Prints version information

OPTIONS:
    -u, --urls <urls>...    The url(s) to start crawling from
```
